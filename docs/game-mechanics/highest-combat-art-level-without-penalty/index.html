Highest Combat Art Level Without Penalty (HCALWP)

⚠️ Work in progress

This penalty is applied only to the playable characters, not the creatures.

Each aspect has its own HCALWP.

HCALWP = 1.75 + CharacterLevel * 0.25 + FocusSkillValue * 0.0125

FinalCombatArtLevel = (2 * HCALWP * BaseCombatArtLevel) / (HCALWP + BaseCombatArtLevel)
