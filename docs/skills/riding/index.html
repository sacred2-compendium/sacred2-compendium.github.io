Riding Mastery

Grants the ability to ride better horses, reduces the regeneration penalty and increases the movement speed while mounted. Additionally, special abilities may be unlocked.

  • Mount Combat Arts +X%
  • Maximum Horse Level +X
  • Increased Mounted Run Speed +X%

Item Bonuses Governed By Skill

  • Mounts commonly feature item bonuses such as Block chance (Riding) or Chance to Reflect (Riding).
  • These bonuses remain locked until the governing skill is learned or mastered, as specified by each bonus.
  • Unlike regular item bonuses, skill-governed bonuses scale with the current skill level rather than item level, reaching maximum potential at skill level 250.
  • Additionally, while regular item bonuses use the item's difficulty level to determine their value range, skill-governed bonuses instead use the difficulty level of the current game session.
  • As a result, these bonuses adjust dynamically based on both the current skill level and difficulty level.

Skill bonuses

Mount Combat Arts +X%

  • Increases the damage of horse combat arts.
  • Stacks additively with other percentage-based damage bonuses.

Maximum Horse Level +X

  • This bonus is non-functional. It has no effect on the mount vendors or the mounts themselves.

Increased Mounted Run Speed +X%

  • Increases Run Speed while mounted.
  • Most mounts are already near the Run Speed cap of 250, so there might not be much of a benefit from this bonus. Still, having more speed is beneficial in case of debuffs.

Regeneration Time Penalty from Mounts -X% (Hidden bonus)

  • Horses and special mounts come with a regeneration time penalty, similarly to buffs. This penalty is applied only while the character is mounted.
  • This penalty depends on:
    • The mount's type.
    • The ratio between the character level and mount level.
  • This bonus reduces the regeneration time penalty of mounts by the listed percentage.
