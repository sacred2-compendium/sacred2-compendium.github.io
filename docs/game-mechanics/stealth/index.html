Stealth

Stealth is a game mechanic that can prevent an opponent from directly targeting the stealthed character. Whether targeting is prevented depends on a comparison between the stealthed character's Stealth value and that opponent's individual Stealth threshold.

  • If the Stealth value is equal to or higher than the opponent's threshold, the stealthed character cannot be directly targeted by that opponent.
  • If the Stealth value is lower than the opponent's threshold, the stealthed character can be targeted normally.

Behavior Differences

  • Player with Stealth: Hostile NPCs that fail the detection check will not react to the player and remain unaware of their presence.
  • Enemy with Stealth: If the player fails the detection check, the enemy remains visible but cannot be directly targeted with the mouse. However, the player may still use area-of-effect (AoE) abilities or choose to avoid the enemy.

Sources of Stealth Value

In the unmodified game, Stealth value is granted exclusively by the following combat arts:

  • Shadow Warrior
    • Shadow Veil
    • Shadow Veil (Buff)
  • High Elf
    • Cascading Shroud (requires the Inconspicuous modification)
    • Shadow Step (requires the Vanish modification)

The Stealth value granted by these combat arts is further increased by the damage bonus from the relevant Aspect Lore skill.

Visual Effects

The above-mentioned combat arts also apply visual effects (such as transparency or shadowing), but these are purely cosmetic. Visual appearance does not indicate whether a character has successfully avoided detection.

Formula for Stealth Threshold

The following formula is used to calculate a character's Stealth threshold:

StealthThreshold = FLOOR(2 * CreatureLevel * (1 + EnhancedPerceptionSkillValue * 0.001))
