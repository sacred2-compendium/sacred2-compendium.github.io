Familiarity Bonus

⚠️ Work in progress

All creatures, including the playable characters, are categorized into "subfamilies".

  • SUBFAM_LIFE_PLANT (Plant)
  • SUBFAM_LIFE_ANIMAL (Wild Animal)
  • SUBFAM_LIFE_ANIMAL_ATMO
  • SUBFAM_LIFE_HUMAN (Human)
  • SUBFAM_LIFE_HE_HUMAN
  • SUBFAM_LIFE_BARBARIAN (Humanoid Monster)
  • SUBFAM_LIFE_HIGHELF (High Elf)
  • SUBFAM_LIFE_DWARF
  • SUBFAM_LIFE_ORC (Orc)
  • SUBFAM_LIFE_DRYAD (Dryad)
  • SUBFAM_LIFE_MECHANICAL (Mechanical)
  • SUBFAM_LIFE_UNDEAD (Undead)
  • SUBFAM_LIFE_T_MUTATION (T-Mutant)
  • SUBFAM_LIFE_INSECT (Insect)
  • SUBFAM_LIFE_REPTILE
  • SUBFAM_LIFE_DRAGON (Dragon)
  • SUBFAM_LIFE_DEMON (Demon)
  • SUBFAM_LIFE_ENERGY (Energy)
  • SUBFAM_LIFE_MONSTER (Monster)
  • SUBFAM_TRAP_TURRET (Trap)
  • SUBFAM_TRAP_SWITCH (Trap)
  • SUBFAM_TRAP_AREA (Trap)
  • SUBFAM_TRAP_PHYSICS (Trap)
  • The Dragon Mage's subfamily is Human.
  • The Dryad's subfamily is Dryad.
  • The High Elf's subfamily is High Elf.
  • The Inquisitor's subfamily is High Elf.
  • The Seraphim's subfamily is Human.
  • The Shadow Warrior's subfamily is Undead.
  • The Temple Guardian's subfamily is Mechanical.

This bonus affects only basic attacks and weapon-based combat arts, not spells.

The number of kills can be found in the sacred2stats file (next to each save file).

The formula is:

ValueOfBonus = 1032 * Kills / (1032 + Kills)

DamageBonus = ValueOfBonus / 10 (affected by precision loss and rounded down to one decimal place)
