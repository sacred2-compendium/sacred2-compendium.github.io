Hitpoint Bonus from Attributes

Vitality increases maximum life hitpoints.

Willpower increases maximum energy shield hitpoints.

The formula for both is:

Multiplier = 10 / ((10 + CreatureLevel) * LifeAttribut)

Where:

  • Multiplier represents the hitpoint bonus per attribute point.
  • LifeAttribut is a parameter from balance.txt. The default value in the unmodded game is 25. Setting it to 0 is the same as setting it to 1.
