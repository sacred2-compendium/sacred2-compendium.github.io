Base Hitpoints

The base hitpoints are used in the formulas for both life hitpoints and energy shield hitpoints.

The formula is:

BaseHP = ((2 * LifeStep + CreatureLevel + 1) * CreatureLevel + 2 * LifeBase) / 4

Where:

  • LifeStep is a parameter from balance.txt. The default value in the unmodded game is 19.
  • LifeBase is a parameter from balance.txt. The default value in the unmodded game is 130.

For the playable characters, the above formula is used as-is.

For creatures, the formula is expanded as follows:

NewBaseHP = BaseHP * 1000 / LifeQuotientMT

Where:

  • LifeQuotientMT is a parameter from balance.txt. The default value in the unmodded game is {2500,800,125,125}. Each value in the tuple corresponds to the respective "monstertype": normal, elite, boss, unknown.
