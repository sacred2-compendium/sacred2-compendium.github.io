Consumable Trophies

Consumable trophies become usable only after the respective governing skill is learned.

The bonuses that these trophies provide are also governed by the respective skill. Their values adjust dynamically based on both the current skill level and difficulty level.

The base duration is 20 seconds. This is determined by the parameter DurationBoost from balance.txt. The default value in the unmodded game is 2000.
