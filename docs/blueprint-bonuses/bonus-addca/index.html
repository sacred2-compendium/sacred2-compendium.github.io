BONUS_ADDCA

The type of this bonus is BONUS_ADDCA.

The formula is:

X = FLOOR(TotalValue / 10)

All combat arts +X

Increases the level of all combat arts by X.

The values of spez and spez2 must be left blank.

Combat Art +X

Increases the level of a specific combat art by X.

The values of spez and spez2 must be set to the combat art's aspect and name.
