Spell Damage Bonus from Intelligence

⚠️ Work in progress

LevelDivisor = MIN(BaseCombatArtLevel, CEIL(CharacterLevel / 3))

IntelligenceMultiplier = PL32(0.001 * FLOOR(attrSdam_fact * 0.001 * FLOOR(25 * (1 / SpellBaseDamage) * FLOOR(40 * Intelligence / LevelDivisor))))

Where:

  • BaseCombatArtLevel is the level of the combat art before applying the penalty. Includes consumed runes and equipment bonuses.
  • SpellBaseDamage is a parameter from balance.txt. The default value in the unmodded game is 25.
  • attrSdam_fact is a parameter from balance.txt. The default value in the unmodded game is 334.
  • PL32() Apply precision loss from 32-bit floats and round down to 3 decimal places.

IntelligenceDamageBonus = Damage * IntelligenceMultiplier

The intelligence multiplier is applied to spell damage, spell damage-over-time (DoT) and essence damage.
