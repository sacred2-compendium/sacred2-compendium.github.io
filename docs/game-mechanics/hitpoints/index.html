Hitpoints

⚠️ Work in progress

IntermediateHP = BaseHP * (1 + Vitality * VitalityMultiplier + Constitution * ConstitutionMultiplier) / LifeQuotient

Where:

  • BaseHP - Base Hitpoints.
  • VitalityMultiplier - Hitpoint Bonus from Attributes.
  • ConstitutionMultiplier - Hitpoint Bonus from Constitution.
  • LifeQuotient applies only for non-player characters (NPC). Formula below.

LifeQuotient = LifeQuotientMT / 1000

Where:

  • LifeQuotientMT is a parameter from balance.txt. The default value in the unmodded game is {2500,800,125,125}. Each value in the tuple corresponds to the respective "monstertype": normal, elite, boss, unknown.

MaxHP = MAX(1, IntermediateHP - WoundPenalty + FlatBonus + PercentageBonus + LifeFactorBonus)

Where:

  • LifeFactorBonus applies only for non-player characters (NPC). Formula below.

WoundPenalty = IntermediateHP * WoundMultiplier

Where:

  • WoundMultiplier depends on the wounds that are inflicted on the character.
    • No wounds: 0
    • Deep Wounds OR Serious Open Wounds OR Deadly Wounds: 0.2
    • (Deep Wounds OR Serious Open Wounds) AND Deadly Wounds: 0.4

StepSize = MAX(1, IntermediateHP) * 0.001

FlatBonus = FLOOR(BONUS_LIFEPOITS / StepSize) * StepSize

Where:

  • BONUS_LIFEPOITS is the total value of this bonus (Max. Hitpoints +X).

PercentageBonus = IntermediateHP * BONUS_LIFE_REL_MOUNTED * 0.001

Where:

  • BONUS_LIFE_REL_MOUNTED is the total value of this bonus (Rider's Max. Hitpoints +X%).

LifeFactorBonus = IntermediateHP * (LifeFactorDiff * 0.001 - 1) (NPC only)

Where:

  • LifeFactorDiff is a parameter from balance.txt. The default value in the unmodded game is {1000,1400,2000,3000,4000}. Each value in the tuple corresponds to the respective difficulty level: bronze, silver, gold, platinum and niobium.
