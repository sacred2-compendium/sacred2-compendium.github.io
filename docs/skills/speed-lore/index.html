Speed Lore

Speed Lore

Increases run speed as well as attack and defense values with all weapons.

  • Attack and Defense Value +X
  • Run Speed +X%

At Level 75: Speed Lore Mastery

Further increases run speed.

Speed Lore Mastery

Increases run speed as well as attack and defense values with all weapons.

  • Attack and Defense Value +X
  • Run Speed +X%

Skill bonuses

Attack and Defense Value +X

  • Functionally equivalent to Combat Values +X%.
    • It is a percentage-based bonus, not a flat bonus. And the decimal part is not visible in the tooltip.
  • Increases total Combat Values by a percentage of base Combat Values.
  • Stacks additively with other sources of Attack Value +X% and Defense Value +X%.

Run Speed +X%

  • Increases total Run Speed by a percentage of base Run Speed.
  • Stacks additively with other sources of Run Speed +X%.
