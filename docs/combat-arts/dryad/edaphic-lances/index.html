Modification details

Thorns

Adds more thorns to Edaphic Lances.

Density

Decreases distance between thorns, thus causing more damage to nearby opponents.

Stings

Each thorn inflicts more damage.

Barbs

Successful hits have an increased chance to inflict bleeding wounds.

Singe

Thorns inflict additional fire damage.
