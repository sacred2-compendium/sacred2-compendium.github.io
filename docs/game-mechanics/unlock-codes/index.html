Unlock Codes

There are 37 unlock codes available that can be used to obtain special items. In the unmodified game, this is the only method to acquire these items.

To use an unlock code, open the in-game console by pressing `, then enter the following command:

UNLOCK XXXX-XXXX-XXXX-XXXX

  • Replace XXXX-XXXX-XXXX-XXXX with the desired code.
  • Letter case does not matter.
  • Hyphens are optional — the game will accept the code with or without them.

If the command is successful, the corresponding item will appear at the character's location. The item's level is set to the character's level, and its difficulty level is set to the difficulty level of the current game session. The item's tier is set according to its blueprint: most are tier 13, while the shield Grunwald's Protection is tier 12.

Note: Each code can be used only once per character. Used codes are recorded in the character's save file.
