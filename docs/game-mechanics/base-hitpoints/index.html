Base Hitpoints

The base hitpoints are used in the formulas for both life hitpoints and energy shield hitpoints.

The formula is:

BaseHP = ((2 * LifeStep + CreatureLevel + 1) * CreatureLevel + 2 * LifeBase) / 4

Where:

  • LifeStep is a parameter from balance.txt. The default value in the unmodded game is 19.
  • LifeBase is a parameter from balance.txt. The default value in the unmodded game is 130.
