Testa's Band

FAMILY_FORGE, SUBFAM_FORGE_RING, CLF_RING

Testa's Band is a unique ring. The item tier is 13.

This ring has several variants depending on the itemtype:

  • Itemtype ID 11592
    • Can only be worn by a Seraphim.
  • Itemtype ID 11597
    • Can only be worn by a Temple Guardian.
