Block Chance

The type of this bonus is BONUS_CHANCE_BLOCKDAMAGE.

The formula is:

X = 100 * (TotalValue + 32) / (TotalValue + 1032)

This bonus has the following variations based on the value of spez:

Block chance: Close Combat +X%

The spez of this bonus is BLOCK_CLOSECOMBAT.

Grants a chance to negate damage from melee weapon attacks, which includes basic attacks and weapon-based combat arts.

Block chance: Combat Arts +X%

The spez of this bonus is BLOCK_SPELL.

Grants a chance to negate spell damage of combat arts. This does not include spell damage-over-time (DoT) or other effects.

Block chance: Projectiles +X%

The spez of this bonus is BLOCK_MISSILE.

Grants a chance to negate damage from ranged weapon attacks, which includes basic attacks and weapon-based combat arts.

Block chance: Root +X%

The spez of this bonus is BLOCK_ROOT.

Block chance: Stun +X%

The spez of this bonus is BLOCK_STUN.

Additional Notes

The bonus Block chance: Knock back +X%, while related in name, is actually a different bonus.
