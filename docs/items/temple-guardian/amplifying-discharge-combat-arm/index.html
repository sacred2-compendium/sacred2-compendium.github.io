Amplifying Discharge Combat Arm (Temple Guardian)

This item is special and cannot be obtained normally. It is equipped automatically while the Temple Guardian has Amplifying Discharge in the currently active combat art slot, replacing the Default Combat Arm.

  • The level of the item is set to the character's level.
  • The difficulty level of the item is set to the difficulty level of the current game session.
  • The tier of the item is set to the minimum allowed by the blueprint, which is 0.

This item occupies the "gloves" slot, which the Temple Guardian cannot normally use. It has no effect on regeneration time.
