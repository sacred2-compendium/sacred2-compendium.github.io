Alchemy

Alchemy

Enhances the effect of all potions. The alchemist will also be able to use special consumable items.

  • Potion Effect X%

At Level 75: Master Alchemist

Further enhancement of all potions' effects. Allows the use of new items.

Master Alchemist

Significantly enhances the effects of all potions. Enables the alchemist to detect hidden attributes of usable items.

  • Potion Effect X%

Consumable Trophies

  • Enables the use of consumable trophies governed by Alchemy.

Item Bonuses Governed By Skill

  • The above-mentioned consumable trophies grant bonuses such as Speed (Alchemy) or Chance for critical hits (Alchemy).
  • These bonuses remain locked until the governing skill is learned or mastered, as specified by each bonus.
  • Unlike regular item bonuses, skill-governed bonuses scale with the current skill level rather than item level, reaching maximum potential at skill level 250.
  • Additionally, while regular item bonuses use the item's difficulty level to determine their value range, skill-governed bonuses instead use the difficulty level of the current game session.
  • As a result, these bonuses adjust dynamically based on both the current skill level and difficulty level.

Skill bonuses

Potion Effect X%

  • The listed value uses 100% as the base. For example, 105% means a 5% increase over the base value.
  • Increases the potency of Health Potions, Recovery Elixirs, and Roast Joint Trophy.
  • Increases the duration of all other potions and all consumable trophies.

Additional notes

  • Roast Joint Trophy is affected twice by this skill. As a governing skill, Alchemy determines the base value of the bonus. That same value is then increased by the Potion Effect +X% bonus.
