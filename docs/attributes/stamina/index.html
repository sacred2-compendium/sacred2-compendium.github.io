Stamina

Decreases the general regeneration time of combat arts.

Attribute Bonuses

Average Combat Art Regeneration Time X%

  • The regeneration time of combat arts and regeneration time penalty imposed by buffs naturally decrease as the character's level goes up.
  • Stamina further reduces them.
  • Both character level and Stamina contribute to the same multiplier as shown in the following formula:
    • Multiplier = 200 / (147 + CharacterLevel * 3 + Stamina * 2)
  • From this formula, we can infer that one character level has the same effect as 1.5 points of Stamina.
