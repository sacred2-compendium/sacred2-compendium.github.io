Modification details

Bleed

An attack which inflicts damage and causes bleeding (damage over time).

Tenacity

An attack that will slightly recharge all combat arts.

Lacerate

The opponent suffers from a deep wound that will decrease the base hitpoint value for a brief period.

Startle

Chance to leave the targeted opponent stunned for a brief period.

Draw Life

Draws life from the opponents.

Judgement

Increases chance for critical hits.
