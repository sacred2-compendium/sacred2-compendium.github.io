Item Bonuses Governed By Skill

  • Low- and mid-tier pole arms commonly feature item bonuses such as Combat Values (Pole Arms) or Chance that opponents cannot evade attacks (Pole Arms Mastery).
  • These bonuses remain locked until the governing skill is learned or mastered, as specified by each bonus.
  • Unlike regular item bonuses, skill-governed bonuses scale with the current skill level rather than item level, reaching maximum potential at skill level 250.
  • Additionally, while regular item bonuses use the item's difficulty level to determine their value range, skill-governed bonuses instead use the difficulty level of the current game session.
  • As a result, these bonuses adjust dynamically based on both the current skill level and difficulty level.

Skill bonuses

Attack Value +X

  • Increases the character's base Attack Value by the listed amount.
  • Stacks additively with other sources of Attack Value +X.

Attack Speed +X%

  • Increases the Attack Speed of basic attacks.
  • Stacks additively with other sources of Attack Speed +X%.

Execution Speed +X%

  • Not shown in the skill tooltip, but confirmed via testing.
  • Has the same progression as the Attack Speed +X% bonus.
  • Increases the Execution Speed of combat arts influenced by Tactics Lore.
  • Stacks additively with other sources of Execution Speed +X%, like Casting Speed +X%.

Item Level without Penalty X

  • Higher-level weapons (and shields) impose a penalty to the Attack Speed of basic attacks.
  • They also impose a penalty to the Execution Speed of combat arts influenced by Tactics Lore.
    • This penalty is half as much as the penalty to Attack Speed.
  • The higher the item level is above the character's, the more severe the penalties.
    • The formula is: Multiplier = MIN(1, ((CharacterLevel + 9) / (ItemLevel + 9))2)
  • This bonus increases the character's effective level (expressed as X) by 0.5 per skill point, thereby reducing the penalties.
    • This does not affect the Maximum Usable Item Level; characters still cannot equip items above this limit.
  • In the tooltip, X is rounded down and capped at Maximum Usable Item Level minus 2. Despite what the tooltip shows, X is not actually rounded down or capped.
  • The reduction is applied only when a relevant weapon is equipped in the currently active weapon slot.
  • When multiple weapons are equipped (e.g., Dual Wield) or a weapon and a shield are equipped, only the highest penalty among them is imposed; these penalties do not stack.
  • Penalties from other sources, such as debuffs, stack additively with the penalty from equipped items.

Chance for Double Hit +X%

  • This bonus relates to the chance of performing a special attack animation that involves an extra hit.
    • In the case of pole arms, this means 2 hits instead of 1.
  • This special attack animation concerns only basic attacks and has a chance to occur even without the respective weapon skill.
  • The skill simply increases the chance for it to occur. Skill mastery is not required, but it will raise the chance even further.
  • Do not confuse with the item bonus Chance for Double Hit +X%.
