Skills

Characters begin without any skills. They can learn up to 10 different skills by using special skill points, earned at levels 2, 3, 5, 8, 12, 18, 25, 35, 50, and 65. When a new skill is learned, it starts at level 1.

Characters also earn regular skill points every time they level up. These are used to increase the levels of skills the character has already learned.

  • Levels 2-4: Characters earn 1 skill point per level (3 × 1 = 3).
  • Levels 5-11: Characters earn 2 skill points per level (7 × 2 = 14).
  • Levels 12-24: Characters earn 3 skill points per level (13 × 3 = 39).
  • Levels 25-49: Characters earn 4 skill points per level (25 × 4 = 100).
  • Levels 50-200: Characters earn 5 skill points per level (151 × 5 = 755).
  • Total skill points earned by level 200: 911. View the full table.

When investing skill points, a learned skill can only be raised up to the character's current level. This limit can be bypassed by using external tools, such as a character editor, or via known exploits. Skill levels increased through invested skill points contribute to skill mastery.

Skill levels can also be increased through bonuses from equipment or combat arts. These bonuses bypass the level restriction but do not contribute to skill mastery.
