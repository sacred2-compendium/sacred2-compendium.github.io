Hitpoint Bonus from Constitution

Constitution increases maximum life hitpoints.

The formula is:

Multiplier = mean_value / (10 * LifeAttribut * (CreatureLevel + AdjustConstitutionFactor))

Where:

  • Multiplier represents the hitpoint bonus per skill point.
  • LifeAttribut is a parameter from balance.txt. The default value in the unmodded game is 25. Setting it to 0 is the same as setting it to 1.
  • AdjustConstitutionFactor is a parameter from balance.txt. The default value in the unmodded game is 5. Setting it to 0 is the same as setting it to 5.
